QR Code web-component demo

Sample web-component made with Stencil

DataURI QR Code

<qr-code
  contents="Hello World"
  output-mode="DataURI"
  style="width: 100px;height: 100px;">
</qr-code>

QR Code with wider margin

<qr-code
 contents="Hello World"
 output-mode="DataURI"
 margin="10"
 style="width: 100px;height: 100px;">
</qr-code>

QR Code with bigger scale (finer image as size is handled by styles)

<qr-code
  class="with-rounded-corners"
  contents="Hello World"
  output-mode="DataURI"
  scale="10"
  style="width: 100px;height: 100px;">
</qr-code>

colorized QR Code

<qr-code
  contents="Hello World"
  color-light="#50550088"
  color-dark="#50005500"
  style="width: 100px;height: 100px;">
</qr-code>

SVG QR Code

<qr-code
  class="with-rounded-corners"
  contents="Hello World"
  output-mode="SVG"
  style="width: 100px;height: 100px;">
</qr-code>
